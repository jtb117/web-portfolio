Projects

Here are a selection of projects I've completed.

  • GroupMe Analysis Bot - GitHub

    Current Features:

    • !all - Mention all users in the chat.
    • !remember {trigger}::{response} - The bot will send a message identical to response when a message identical to trigger is sent.
    • !forget {trigger} - The trigger will no longer prompt response
    • !triggers - Returns a list of saved triggers.
    • Unrecognied command - Bot will respond "Huh?" to indicate the command is not recognized.

  • PGH Healthy Ride Bike Data Analysis - GitHub


    • Assignment was to conduct analysis of PGH Healthy Ride Bike Data
    • Jupyter Notebook file and and datasets included
    • Primarily focused on data cleaning/wrangling in Pandas
    • For example, the rebalancing (manually moving bikes from one station to another) was not explicitly mentioned in data set and had to be calculated.

  • Housing Price Data Analysis and Predictions - GitHub


    • Assignment was to predict housing prices in test set given 14 potential predictors using a training set.
    • Conducted entirely in R
    • Includes: data exploration, data cleaning, model comparison, and model selection
    • Written technical as well as non-technical reports included
